Um código html consistente, limpo e organizado torna mais facil para outras pessoas ler e entender seu código.

  1. Sempre declare o tipo do documento na primeira linha
  2. Sempre use nome de elementos em minusculos
  3. Sempre fechar todas as tags sejam elas vazias ou não
  4. Sempre usar nome de atributos em minusculos
  5. Sempre citar os valores dos atributos e dentro de aspas
  6. Sempre especificar o texto alternativo, altura e a largura da imagem
  7. Nunca use espaço entre o sinal de igualdade.
  8. Evitar linhas de codigos longas.
  9. Nunca adicione linhas em brancos e recuos sem razão.
  10. Nunca ignorar o elemento title, pois é muito importante ter ele.
  11. Torne o titulo mais preciso e significativo possivel
  12. Sempre adicione as tags html e body.
  13. Sempre declarar head
  14. Sempre declarar a tag meta com atributo viewport.
  15. Sempre adicione folha de styles no head
  16. Mesmo do item anterior serve para outros arquivos
  17. Acessando elementos HTML com JS pelo getElementByXXXX
  18. Sempre usar nomes de arquivos em minusculos para evitar problema em servidores
  19. Nome de arquivos padrão index.html (Arquivo pai)
