Label é um rotulo para diverso elementos do formulario, Muito util para leitores de tela. O atibuto for tem que ser igual ao atributo id do elemento input.




Submit

define um botão para enviar os dados do formulário para um manipulador de formulários. O manipulador de formulários geralmente é uma página no servidor com um script para processar dados de entrada. O manipulador de formulários é especificado no atributo de ação do formulário.

Atributo Action

Atributo que define a ação a ser executada quando o formulario é enviado.

Atributo target

O target atributo especifica se o resultado enviado sera aberto em uma nova guia. Valor padão "_self" sera enviado na janela atual.

Atributo method

O atributo especifica o método HTTP ( GET ou POST ) a ser usado ao enviar os dados do formulário. GET os dados ficam visiveis POST os dados se tornam confidencial

Atributo name

Se name for omitido os dados não serão enviados
