Value

Value atributo especifica uma valor inicial para um campo de entrada

Readonly

Readonly especifica que um campo de entrada é somente leitura, o campo de entrada não pode ser alterada

Disabled

O atributo especifica que um campo deve ser desativadom inutilizavel e não clicével

Size

Atributo especifica a largura visivel em caracteres de uma campo de entrada. Padrão é 20.

Maxlenght

Esse atributo especifica a quantidade maxima de caracteres insirido.

Max e Min

Esse atributo define um tamanho de um intervalo de valores.

Multiple

Atributo de entrada especifica que a entrada pode aceitar multiplos valores

Pattern

O atributo especifica um expressão regular com qual o valor do campo de entrada é verificado quando formulario é enviado.

Placeholder

especifica uma dica curta que descreve o valor esperado no campo de entrada.

Required

especifica que um campo deve ser preenchido obrigatorio antes de ser enviado.

Step

Atributo usado para especifica os intervalos de numeros em um campo de entrada

Autofocus

Atributo usado para quando a página de o reload ou carregue, o campo seja focado.

Image width e height

Width e height especifica a altura e a largura de uma imagem carregada.




List

Atributo list cria um campo no formato de lista para a seleção das opções

Autocomplete

Atributo Autocomplete especifica se o formulario ou campo deve ter preenchimento automatico.
