URL

URL -> Localizador Uniforme de Recursos. Pode ser o endereço de iP ou nome do dominio. Utilizado para endereçar um documento. Um endereço tem que seguir essas regras: scheme://prefix.domain:port/path/filename

  1. scheme: define o tipo de serviço da internet (http ou https)
  2. prefix: define um prefixo de dominio (padrão http é www)
  3. domain: define o nome do dominio da internet (google.com)
  4. port: define o numero da porta do host (padão http é 80)
  5. path: define uma caminho no servido omitindo o diretório raiz do site
  6. filename: define o nome de um documento ou recurso (index.html)

A codificação de URL substitui caracteres não ASCII por um "%" seguido por dígitos hexadecimais. URLs não podem conter espaços. A codificação de URL normalmente substitui um espaço por um sinal de mais (+) ou% 20.
